# **App Name**: BayanihanAgri Insights

## Core Features:

- Farmer Registration & Login: Secure registration and login for farmers using email/password authentication.
- Farm Record Input: Farmers can input data related to their farms: crop type, planting date, area, expenses (seeds, fertilizer, labor costs in ₱), inputs used, and harvest quantity.
- Farm History View: Display historical farm records with filtering and sorting options.
- Profit/Loss Calculation: Calculate profit/loss for each harvest based on entered expenses and harvest quantity, displayed in ₱.
- Yield Prediction: AI tool for predicting yield based on historical data, crop type, planting date, and inputs. Uses a simple regression model.
- Cost vs. Profit Analysis: Provide insights into cost trends and profit margins over time.
- Admin Dashboard: Web dashboard for admins to view aggregated farmer data, analytics, and generate reports.

## Style Guidelines:

- Primary color: Warm green (#77DD77), reminiscent of rice paddies and new growth.
- Background color: Off-white (#FAF9F6), providing a clean and readable surface.
- Accent color: Earthy brown (#B19470), suggesting the soil and the hardworking nature of farmers.
- Headline font: 'Poppins' sans-serif for a clean, modern, and readable feel; Body font: 'PT Sans' sans-serif for comfortable readability.
- Use clear, representative icons for crop types, expenses, and other data points. Ensure icons are culturally relevant to the Philippines.
- Mobile app: Clean, single-column layout optimized for easy data input. Web app: Dashboard layout with charts, summaries, and data tables.
- Subtle animations to indicate loading states and successful data saving.
